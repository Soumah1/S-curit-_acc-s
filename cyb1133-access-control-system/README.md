# CYB1133 - Système de contrôle d’accès sécurisé

## Description
Ce projet consiste à concevoir et implémenter un système de contrôle d’accès basé sur les modèles DAC, RBAC et MAC, avec un mécanisme de double authentification (2FA).

## Fonctionnalités
- Gestion des rôles (RBAC)
- Contrôle par niveaux de sécurité (MAC)
- Délégation d’accès (DAC)
- Authentification OTP (2FA)
- Fonction centrale : can_access()

## Structure
- docs/ : rapport et modélisation
- src/ : code source
- tests/ : scénarios
- data/ : données simulées
